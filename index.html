少年中国说

故今日之责任，不在他人，而全在我少年。
少年智则国智，少年富则国富，少年强则国强，少年独立则国独立，少年自由则国自由，少年进步则国进步，
少年胜于欧洲，则国胜于欧洲，少年雄于地球，则国雄于地球。红日初升，其道大光；
河出伏流，一泻汪洋；
潜龙腾渊，鳞爪飞扬；乳虎啸谷，百兽震惶；
鹰隼试翼，风尘翕张；奇花初胎，矞矞皇皇；
干将发硎，有作其芒；
天戴其苍，地履其黄；纵有千古，横有八荒；
前途似海，来日方长。美哉，我少年中国，与天不老！壮哉，我中国少年，与国无疆！

关于毒奶粉的外网架设

需要准备以下条件：
云服务器（是服务器，不是虚拟主机，且至少1G1H）
预装centos5.8或5.11系统（5系列启动速度快，但yum源缺少，下面会提到）
需要软件Xhell、WinSCP、navicat
DNF服务端

一、买服务器
推荐使用阿里云，腾讯云理论上也可以，但上次我没弄好，也懒得深究
其他云没试过，小伙伴们可以自己尝试下
购买阿里云的小伙伴买的时候选自定义购买
计费方式：包年包月
地域：随便选，哪个便宜选哪个
网络：默认的来，不用动
实例：选择其他实例规格
vcpu:1 vcpu
内存：1GB
直接点确定即可
公网带宽：按固定带宽1Mbps
镜像：点开市场镜像——从镜像市场选择
在弹出的页面搜索CentOS 5.8，然后直接点使用即可
储存：按默认来
购买量：土豪们，展现你们财力的时候到了，任性吧
安全设置：设置密码，登录名默认为root,密码随意设置，
实例名称也随意
最后点立即购买即可

以下我将采用阿里云的vps演示

二、下载软件和服务端
软件百度自己搜，都能找到
链接：https://pan.baidu.com/s/1c3cfCOg 密码：h1uu

三、正式安装

步骤简写了哈，一步一步来工程量太大了，脑壳疼


用WinSCP远程登录阿里云的服务器
修改/etc/yum.repos.d/CentOS-Base.repo文件的内容为：

[base]
name=CentOS-$releasever - Base
failovermethod=priority
baseurl=http://vault.centos.org/5.8/os/x86_64/
gpgcheck=1
gpgkey=http://vault.centos.org/5.8/os/x86_64/RPM-GPG-KEY-CentOS-5

#released updates
[updates]
name=CentOS-$releasever - Updates
failovermethod=priority
baseurl=http://vault.centos.org/5.8/updates/$basearch/
gpgcheck=1
gpgkey=http://vault.centos.org/5.8/os/x86_64/RPM-GPG-KEY-CentOS-5

#additional packages that may be useful
[extras]
name=CentOS-$releasever - Extras
failovermethod=priority
baseurl=http://vault.centos.org/5.8/extras/$basearch/
gpgcheck=1
gpgkey=http://vault.centos.org/5.8/os/x86_64/RPM-GPG-KEY-CentOS-5

#additional packages that extend functionality of existing packages
[centosplus]
name=CentOS-$releasever - Plus
failovermethod=priority
baseurl=http://vault.centos.org/5.8/centosplus/$basearch/
gpgcheck=1

gpgkey=http://vault.centos.org/5.8/os/x86_64/RPM-GPG-KEY-CentOS-5

#contrib - packages by Centos Users
[contrib]
name=CentOS-$releasever - Contrib
failovermethod=priority
baseurl=http://vault.centos.org/5.8/contrib/$basearch/
gpgcheck=1

gpgkey=http://vault.centos.org/5.8/os/x86_64/RPM-GPG-KEY-CentOS-5

直接复制粘贴即可，然后保存
另外在当前文件夹内除CentOS-Base.repo的文件都删除
接着将下载好的服务端上传至/root文件夹目录下

用Xhell远程连接阿里云
分别输入以下两条命令：
                      yum clean all
                      yum makecache
接着输入：
         yum install -y mysql mysql-server mysql-devel       #安装数据库

         yum install -y mysql-community-server               #这个包的yum源似乎已经失效用下面的命令代替

                                                      # wget http://repo.mysql.com/mysql-community-release-el5.rpm 

                                                      # ls

                                                      # rpm -ivh mysql-community-release-el5.rpm  

                                                      # ls -1 /etc/yum.repos.d/mysql-community* 

                                                      # yum install mysql-server


以下命令参照和修改自老牛的命令

#关闭防火墙和防火墙自启

service iptables stop
chkconfig iptables off

#安装支持库
                              #个别可能没有安装包可供安装，没关系 
yum -y install glibc.i386
yum -y install xulrunner.i386               
yum -y install libXtst.i386
yum -y install gcc gcc-c++ make zlib-devel


#数据库开机启动

chkconfig mysqld on

#解压下载的文件

tar -zxvf ./DNFServer.tar.gz -C /

3.设置对应权限

#home root mysql文件夹全部权限

chmod -R 777 /home
chmod -R 777 /root
chmod -R 777 /var/lib/mysql
chown -R mysql.mysql /var/lib/mysql/

#重启数据库，如果命令行显示最后一个是绿色的OK，就表示数据库装成功，如果显示的是红色的fail，表示失败，
请自行解决，或者等候我补充

service mysqld restart

4.编译GeoIP

cd /home/GeoIP-1.4.8/
./configure
make clean
make
make check
make install

5.修改对应位置的外网IP
用Navicat打开数据库d_taiwan的db_connect表，把db_ip字段所有的192.168.200.131改成你的外网IP。
连接地址：你服务器的公网ip
账号game  密码uu5!^%jg


#服务端程序里面的用命令改，注意把“你的外网IP”替换成你的外网IP

cd /home/dxf/
sed -i "s/192.168.200.131/这个位置填你外网ip/g" `find . -type f -name "*.tbl"`
sed -i "s/192.168.200.131/这个位置填你外网ip/g" `find . -type f -name "*.cfg"`

#虚拟内存
cd /var/
dd if=/dev/zero of=swapfile bs=1M count=8192
mkswap swapfile 
swapon swapfile
echo /var/swapfile swap swap defaults 0 0  >> /etc/fstab

服务端架设到此结束
